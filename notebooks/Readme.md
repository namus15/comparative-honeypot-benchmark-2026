# Notebooks Jupyter - Análisis Estadístico

Carpeta con los 6 notebooks para el análisis completo del benchmark honeypots (2026).

- 01-data-cleaning.ipynb: Limpieza y anonimización de logs
- 02-interaction-analysis.ipynb: Interaction-to-Attack Ratio
- 03-fingerprinting-resistance.ipynb: Pruebas de evasión
- 04-resource-consumption.ipynb: Gráficos CPU/RAM
- 05-cost-analysis.ipynb: Modelo TCO/ROI Monte Carlo
- 06-statistical-tests.ipynb: Friedman, Wilcoxon, Cliff’s delta

En desarrollo (enero 2026). Se completarán con datos reales de la fase principal.
Requisitos: Python 3.11+, pandas, scipy, statsmodels, matplotlib.
